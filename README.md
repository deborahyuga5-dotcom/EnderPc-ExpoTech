## Sobre o Projeto

O EnderPC é um projeto inspirado no Baú do Fim do Minecraft. O objetivo foi desenvolver um computador temático utilizando um baú decorado e aplicar conhecimentos de hardware, sistemas operacionais e virtualização.

## Materiais Utilizados

- Baú de madeira
- EVA colorido
- Cola quente
- Computador
- Oracle VirtualBox
- Sistema Operacional Lubuntu

## Configuração do Computador

- Sistema Operacional: Lubuntu Linux
- Processador: Intel Core i7-2600 @ 3.40 GHz
- Arquitetura: 64 bits
- Núcleos: 4
- Threads: 8
- Memória RAM: 16 GB
- Armazenamento: 120 GB
- Placa de Vídeo: Intel HD Graphics 2000

## Desenvolvimento

Foi criada uma máquina virtual utilizando o Oracle VirtualBox. Durante o projeto foi realizada a instalação do sistema operacional Lubuntu e a execução de um jogo desenvolvido em Python utilizando a biblioteca Pygame.

## Resultado

O projeto demonstrou o funcionamento de uma máquina virtual, a execução de aplicações em Linux e a integração entre hardware e software em um projeto inspirado no universo Minecraft.
